# 🚀 Installation Guide

## Quick Install

### NPM (Universal)
```bash
npm install -g fortune-cookie-cli
```

### OpenClaw / ClawHub
```bash
clawhub install fortune-cookie-cli
```

### Claude Code Plugin
```bash
claude plugin install fortune-cookie-cli
```

### Codex CLI
```bash
codex plugin install fortune-cookie-cli
```

## Usage

After installation, simply run:
```bash
fortune
```

### Personalities
```bash
fortune --personality zen          # Calm wisdom
fortune --personality hype         # Over-the-top encouragement
fortune --personality roast        # Brutal honesty
fortune --personality nostradamus  # Predictions
fortune --personality shakespeare  # Dramatic flair
```

## No Configuration Required

Works out of the box. Zero setup needed.
