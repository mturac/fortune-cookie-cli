# Fortune Cookie Skill

Get a coding fortune every time you need guidance.

## Usage

```bash
# Get a random fortune  
fortune

# Get roasted
fortune --personality roast

# Get hyped  
fortune --personality hype

# List personalities
fortune list
```

## Personalities

- zen - Calm wisdom
- hype - Over-the-top encouragement
- roast - Brutal honesty
- nostradamus - Predictions
- shakespeare - Dramatic flair

## Installation

```bash
npm install -g fortune-cookie-cli
```
